# Developer Application Programming Interface (API)

This document provides a reference of the public Eask API, which you may use in
your projects and extensions to Eask.

```elisp
(eask-start
  (message "Hello World!~"))
```

```elisp
(eask-with-verbosity 'debug
  (message "Hello World!~"))
```
